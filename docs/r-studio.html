5.1 Data types

  • Numeric
a=23.5
class(a) #check the data type of a 
## [1] "numeric"
  • Integers
a=23L #add L to show it is an integer
class(a)
## [1] "integer"
  • Characters
a="DNA"
class(a)
## [1] "character"
  • Logical
a=TRUE #logical data types is either TRUE or FALSE only
class(a)
## [1] "logical"
  • Complex,
a=2+3i # Complex data types have 'i' at the end of each number
class(a)
## [1] "complex"
  • Raw
a=charToRaw("DNA")
print(a)
## [1] 44 4e 41
class(a)
## [1] "raw"
# convert back to character 
b=rawToChar(a)
class(b)
## [1] "character"
